# Hyperion: Graph-Driven Enterprise Data Mesh Drift & Chaos Simulator

Hyperion is an enterprise-grade, offline-first simulation platform that models complex distributed data architectures as dynamic directed graphs.

## Features
- Interactive Topology Studio
- Fault Injection Control Center
- Live Stream Event Viewer
- Automated Policy Sandbox
- Comparative Historic Run Replayer

## Quickstart

Run with docker-compose:
```bash
docker-compose up --build
```

Access the UI at `http://localhost:3000` and API at `http://localhost:8000`.
